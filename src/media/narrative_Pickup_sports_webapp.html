PICKUP SPORTS

This is a perfect application for sports lovers who want to connect with more players and make their experience more happening. A user can sit back home and can choose to join multiple categories of sports played at different venues.

Few major features of this this hustle fee application include:

Different Event categories:

This application not only provide many options to play games but gives user an option to watch an event or attend a workshop.

Different Sports Categories:

The application is not confined to just one sport. Rather people can choose from variety of sports categories including Football, Tennis, Cricket, Chess etc.

User can create events:

Any user who joined the website can also create an event if they are hosting one. That event will be open to other users to join.

Easy access to joined events:

User can easily check the number of events he has joined any point of time.
